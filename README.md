BrewMaster Coffee Shop
Overview

BrewMaster is a modern coffee shop website built with React. It allows users to browse coffee products, place online orders, and see details of their purchases. The website also includes an admin panel for managing orders, inventory, and customer interactions. This project is a resume-ready portfolio project demonstrating frontend development skills, React knowledge, and basic e-commerce functionality.

Features

- Browse coffee products with images and descriptions

- Add products to the cart and place orders

- View order details and status

- Admin panel for managing orders, updating status, and inventory

- Responsive design for mobile and desktop
  

Tech Stack

Frontend: React, HTML, CSS, JavaScript

Backend: Firebase(Authentication, Firestore)


Usage

As a user: Browse coffee items, add to cart, place orders.

As an admin: Log in to the admin panel to view orders, update statuses, and manage products.
